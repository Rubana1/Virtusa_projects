# Online Quiz & Assessment System (Java Project)

## Problem Statement
Educational platforms require systems to conduct quizzes and evaluate users efficiently.  
This project provides a simple and interactive quiz system with automatic evaluation.

## Objectives
- Allow users to take quizzes
- Automatically evaluate answers
- Display results with performance analysis

## Features
- Admin can add questions and answers
- Multiple-choice quiz system
- Timer-based quiz
- Automatic score calculation
- Result summary with performance feedback

## How It Works

### 1. Question Setup
- Admin adds questions with options and correct answers

### 2. Quiz Execution
- Questions are displayed one by one
- User selects answers

### 3. Timer Functionality
- Countdown timer runs during quiz
- Quiz ends automatically when time is up

### 4. Auto Evaluation
- Answers are checked instantly
- Score is calculated

### 5. Result Analysis
- Displays:
  - Score
  - Percentage
  - Performance (Excellent / Good / Needs Improvement)

## Technologies Used
- Core Java
- OOP Concepts (Classes, Objects)
- Collections (ArrayList)
- Timer (java.util.Timer)
- Scanner (User Input)

## How to Run

1. Open terminal / command prompt  
2. Compile the program:
javac QuizSystem.java
java QuizSystem
