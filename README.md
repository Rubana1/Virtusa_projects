# Mini Projects Portfolio

This repository includes projects completed as part of my Virtusa assignment, demonstrating practical skills in Java, Python, and SQL.

## Projects Overview

### 1. SafeLog Password Validator (Java)
A password validation system that enforces security rules such as minimum length, uppercase letters, and digits. It provides specific feedback and ensures strong password creation.

 Project Link: [https://github.com/Rubana1/Virtusa_projects/tree/main/PasswordValidator]


### 2. Online Quiz & Assessment System (Java)
An interactive quiz system with multiple-choice questions, timer functionality, automatic scoring, and performance analysis.

 Project Link: [https://github.com/Rubana1/Virtusa_projects/tree/main/QuizSystem]


### 3. Resume Analyzer & Job Matcher (Python)
A smart tool that analyzes resumes using NLP, matches skills with job descriptions, calculates match score, and suggests improvements.

 Project Link: [https://github.com/Rubana1/Virtusa_projects/tree/main/Resume_Analyzer]


### 4. SocialMedia Content Sanitizer (Python)
A content moderation system that filters banned words, extracts links, and generates reports to ensure safe social media usage.

 Project Link: [https://github.com/Rubana1/Virtusa_projects/tree/main/Content_Sanitizer]


### 5. Hospital Management & Patient Analytics System (SQL)
A database system that manages hospital data and provides insights such as doctor performance, patient visits, and revenue analysis.

 Project Link: [https://github.com/Rubana1/Virtusa_projects/tree/main/Hospital_management]


### 6. Retail Insights Data Analysis (SQL)
A data-driven inventory system that identifies expiring products, detects dead stock, and analyzes revenue contribution for better decision-making.

 Project Link: [https://github.com/Rubana1/Virtusa_projects/tree/main/Retail_insights]


## Technologies Used
- Java (Core Java, OOP)
- Python (NLP, File Handling)
- SQL (MySQL, Data Analysis)
